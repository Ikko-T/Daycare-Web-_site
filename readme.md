## プロジェクト

---

保育園の Web サイトを制作する。

この Web サイト制作は練習のため、架空の保育園サイトである。

名称を「ピーターパン保育園」とする。

Adobe XD を使用してワイヤーフレームとデザインカンプを行う。

HTML と CSS でコーディングを行う。

Web 版のみ（レスポンシブ未対応）
